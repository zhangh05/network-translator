```
└── trace_*.json                         ← generated by audit_trace.sh
```

### Naming Convention

```
{phase}_{descriptive_name}_{YYYY-MM-DD}.{ext}
```

- Phase prefix: e.g. `phase6`, `phase7`, `phase8`
- Descriptive name: snake_case, matches content purpose
- Date: ISO-8601 date of generation
- Extension: `.md` for human-readable, `.json` for machine-readable

### Mandatory Metadata Fields

Every generated JSON report must contain:

```jsonc
{
  "schema_version": "1.0",
  "generated_at": "2026-05-23T10:00:00",
  "commit_hash": "abc123def456",
  "run_id": "phase8c-trace-001",
  // ... content fields
}
```
